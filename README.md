# Project Overview

This repository contains my Digital Literacy Portfolio created as part of the CSE0001 Digital Literacy course. The purpose of this project is to demonstrate an understanding of essential digital skills required for academic, professional, and responsible online engagement.

As a Student Digital Ambassador, this project focuses on building awareness about digital tools, professional online presence, communication etiquette, and cyber safety. Each task in this repository corresponds to one module of the Digital Literacy course and documents my learning and practical implementation.

The portfolio includes an infographic, professional digital profiles, coding platform exploration, email communication samples, and a cybercrime awareness case study. All files, screenshots, and reflections have been organized into clearly structured folders for easy navigation.

# Task 1: Digital Literacy Awareness Infographic

This task involved creating a one-page infographic explaining the concept of digital literacy and its importance for students. The infographic highlights safe internet practices, useful digital tools, and maintaining a professional online presence.

The design was created using an online design tool and exported as an image. The final infographic is available in the **task-1-presentation** folder.

# Task 3: Coding and Collaboration Platforms

This task focused on exploring coding practice platforms and collaborative tools.

Part A included solving a beginner-level coding challenge to gain familiarity with problem-solving platforms.

Part B involved creating a Digital Literacy Awareness Quiz using Google Forms. The quiz includes five questions designed to assess basic digital literacy knowledge.

Screenshots and documentation are available in the **task-3-platforms** folder.

# Task 4: Professional Email and Digital Communication
